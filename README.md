# limestone-python

An api to access trusted token pricing data secured on Arweave and protected by provider's collateral.

## Installation
`pip install limestone`

## Usage

```python
from limestone import get_price

price = get_price("AR")
```

## Data format

```
{
  price: 2.05, //as Float
  updated: '2020-11-03 16:00:00', //as Date
}
```
